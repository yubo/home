# HOME
My working development environment configuration, including shell, tmux, vim, git, ssh, etc.

## install
```shell
git clone git@github.com:yubo/home.git ~/.yubo
cd ~/.yubo && sh install.sh
fc-cache -vf ~/.fonts
vim
:BundleInstall
:q
```
