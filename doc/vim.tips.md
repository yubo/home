# vim tips


##### i
---
块选择
```
	+-----+-----------------------------+
	| di" | 删除双引号内的字符          |
	+-----+-----------------------------+
	| yi" | 复制双引号内的字符          |
	+-----+-----------------------------+
	| vi" | 选取双引号内的字符          |
	+-----+-----------------------------+
```

##### 缩进
---
```
	+---+--------+--------+--------+
	|   | NORMAL | INSERT | VISUAL |
	+---+--------+--------+--------+
	| + | [n]>>  | c-t    | [n]>   |
	+---+--------+--------+--------+
	| - | [n]<<  | -      | [n]<   |
	+---+--------+--------+--------+
```


## 插件

##### EasyMotion
---
```
	+---------------------+--------------+
	| <leader><leader><w> | 向后跳转     |
	+---------------------+--------------+
	| <leader><leader><k> | 向前跳转     |
	+---------------------+--------------+
```

##### 注释 The-NERD-Commenter
---

```
	| remove | <leader>ci | <leader>cu |
	+--------+------------+------------+
```
